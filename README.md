# Web-Store-Nico
# 🛒 Nico Store

Website toko online sederhana berbasis HTML yang menampilkan berbagai produk dengan tampilan modern dan responsif.
Project ini dibuat untuk latihan dan pengembangan website e-commerce sederhana.

---

## 🚀 Fitur Utama

* 🔍 Pencarian produk
* 🛍️ Tampilan produk (grid)
* 📱 Responsive design (menggunakan Tailwind CSS)
* 📞 Integrasi WhatsApp untuk pemesanan
* 📄 Halaman tambahan:

  * Tentang Saya
  * Informasi
  * Cara Belanja
  * Hubungi Kami

---

## 📁 Struktur Project

```
nico-store/
│
├── index.html          (Homepage)
├── tentang-saya.html   (Halaman Tentang)
├── informasi.html      (Informasi toko)
├── cara-belanja.html   (Panduan belanja)
├── hubungi-kami.html   (Kontak)
├── style.css           (Custom CSS)
```

## 📦 Cara Menjalankan Project

1. Download atau clone repository ini
2. Buka folder project
3. Jalankan file:

```
index.html
```

di browser

---

## ☁️ Cara Hosting (Gratis)

### 🔹 GitHub Pages

1. Upload project ke GitHub
2. Masuk ke Settings → Pages
3. Pilih branch `main`
4. Website akan online otomatis

### 🔹 Netlify

1. Buka https://netlify.com
2. Drag & drop folder project
3. Selesai, website langsung online
